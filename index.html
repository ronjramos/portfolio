The Distillery is my first website created which focused on the responsiveness of the website. I have used basic HTML/CSS and the Bootstrap framework. I have also added built in scripts to animate some of the objects in the site. Currently, the website doesn't have any functionalities yet.

The Shoebox is fully functional eCommerce site that I have created from scratch using PHP, MySQL, Javascript, AJAX and Bootstrap. It has the basic CRUD functionalities that can be managed by an Admin. Customers would be able to create a profile and completely checkout with the products that they have purchased.
